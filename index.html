Conheça as ferramentas para projetos de sistemas de infraestrutura urbana da Engedev!

A-REDE

O A-REDE é um software que trabalha embarcado no AutoCAD, ou seja, usa a interface de desenho do AutoCAD para auxiliar no desenvolvimento de projetos de distribuição de água. O objetivo principal desse software é facilitar / automatizar o projeto de redes de distribuição de água.

O A-REDE integra a interface gráfica do AutoCAD com o motor de cálculo do EPANET tornando-se uma ferramenta bastante eficiente para projetos de redes de distribuição de água, basicamente é uma versão do EPANET para AutoCAD.

SESCAD

O SESCAD (Sistema CAD para Esgotamento Sanitário) é um software que trabalha embarcado no AutoCAD, ou seja, usa a interface de desenho do AutoCAD para auxiliar no desenvolvimento de projetos de redes coletoras de esgoto. O objetivo principal desse software é facilitar / automatizar o projeto de redes coletoras de esgoto sanitário.

O grande diferencial do SESCAD em relação a outros softwares é a automatização de cálculos topográficos e hidráulicos com assimilação instantânea ao desenho da rede coletora, ou seja, à medida que vamos desenhando a rede a mesma está sendo dimensionada. Além disso, é possível simular vários cenários alterando dados de ocupação da área, podemos exportar quantitativos para pla nilhas para elaboração de orçamentos.

CADAMS

O Objetivo desta ferramenta é fazer análises de estabilidade de estruturas hidráulicas de concreto por gravidade. Esta ferramenta utiliza funções do AutoCAD para obter dados geométricos dessas estruturas e fazer análise de sua estabilidade, considerando carregamentos hidroestáticos, hidrodinâmicos e Sismos.
